LESSON 7


MANIPULATING CSS CLASS PROPERTIES USING JAVASCRIPT

Coming into this semester I must admit, CSS was my weakest subject. I am happy to say though, after some time learning it more, that I truly overestimated how complicated it is. This section represents not only some cool CSS, but my growth with CSS as a whole. So strap yourself in, we're in for a wild ride.

First, let's look at some simple text manipulation. Let's have some fun with it.

See the Pen CSS Modify by Costigan-Stephen (@costigan-stephen) on CodePen.

We can change any aspect of an object's class using JavaScript using the following syntax:

document.getElementById([elementId]).style.[cssProperty] = [newValue];

In the example above, we manipulated the text using the style.color attribute and the visibility of the text using style.visibility, which allowed us to show or hide elements as need be.

Next, let's change the class of an object with javaScript. This next example will turn an object from a box into a circle. It will do this by changing the CSS class of the object. and trigger an animation in the process. The basic concept here is that utilizing the power of JavaScript, we can do almost anything we want within reason.

We did a little bit of this toward the end of Lesson 7 with the shape changing, but we'll cover it in more detail here. The important thing to notice is that in the above example, the shape transitioned gradually into another shape, it didn't suddenly become a square or a circle. That process was an animation. Our example above only had 2 key frames, but we can add more transition points to reach our desired effect.

in the next example, we will get a little bit fancy with transitions and animations triggered by javascript.
